# Bug-Fight

A Roblox PvP hero-shooter where players transform into bugs and battle across TF2-inspired game modes.
Concept
Players collect bugs of different rarities, equip them for cosmetic outfits and passive bonuses, and fight in class-based team battles. Bugs can be traded between players.

Bug Classes
Assault - High damage frontliners (Mantis, Fire Ant, Wasp)
Aerial - Fast flankers and flag runners (Dragonfly, Moth, Mosquito)
Tank - High HP crowd controllers (Rhino Beetle, Hercules Beetle, Millipede)
Strategic - Utility, traps, and support (Spider, Bombardier Beetle, Stick Insect)

Game Modes
Capture the Flag - Steal the enemy flag and return it to base
Push the Cart - Escort or stop a payload
Control Points - Capture and hold key locations
Hive Defense (planned) - Defend your hive from an attacking swarm

Rarity System
Common → Uncommon → Rare → Epic → Legendary
Higher rarity bugs have stronger passive abilities and are more valuable for trading.
Status
Early concept stage. Built by a small team as a side project.
